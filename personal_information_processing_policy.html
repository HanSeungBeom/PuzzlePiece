개인정보처리방침(Personal Information Processing Policy)

"내 사람들" 개인정보 처리방침은 아래와 같습니다.

법령에 의하거나 별도로 동의하지 않는 한 회원님의 개인정보를 제3자에게 제공하는 일은 결코 없으므로 안심하셔도 좋습니다.

개인정보 처리방침은 항상 공개되어 있으니 지금 당장이 아니더라도 궁금하실때 언제든지 읽어보실 수 있습니다.

"내 사람들" 어플리케이션 내에는 다음과 같은 접근 권한을 사용자에게 요구합니다.

1. 사진/미디어/파일
프로필 사진으로 사용할 사진들을 제공하기 위해서입니다.

2. 카메라(android.permission.CAMERA)
카메라에 접근권한을 요구하는 목적은 프로필 사진으로 사용할 사람의 사진을 직접 촬영하여 등록하는 기능을 제공하기 위해서입니다.

3. 전화상태(android.permission.READ_PHONE_STATE)
등록한 상대로부터 전화가 왔을 때 작성해둔 메모를 화면에 실시간으로 보여주기 위해서입니다.

감사합니다.




"My People" personal information processing policy is as follows.

Your personal information used in the application will be used only within the purpose of using the application for the smooth service. Rest assured that you will never be able to provide your personal information to third parties unless you agree with the statute or agree otherwise. The privacy policy is always open so you can read it at any time, even if it's not right now.

Within the "My People" application, you are prompted for the following access rights:

1. Photo / Media / File
This is to provide pictures to use as profile pictures.

2. Camera(android.permission.CAMERA)
The purpose of requesting permission to access the camera is to provide a function that allows you to directly capture and register a picture of a person to be used as a profile picture.

3. Phone State(android.permission.READ_PHONE_STATE)
This is to display the memo that you made when you received a call from the registered person on the screen in real time.

Thank you.
